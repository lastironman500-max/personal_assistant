# Portfolio Builder

Create a minimum viable portfolio to establish credibility for AI services. Designed for Day 1 setup.

## Inputs Required
- **Services offered**: Which services you sell
- **Target industry**: Who you serve (SaaS, agencies, SMBs)
- **Name/Brand**: Your professional name

## Portfolio Pieces to Create (Day 1)

### Piece 1: Landing Page Sample
- Pick a real company with a weak landing page
- Rewrite their landing page using the `landing-page-builder` skill
- Screenshot or host on Carrd as "before/after"

### Piece 2: Copywriting Sample
- Write a homepage rewrite for a recognizable brand
- Include: hero, features, CTA, social proof sections
- Label as "spec work / concept"

### Piece 3: Pitch Deck Sample
- Create a 10-slide deck for a fictional (or real) startup
- Use the `pitch-deck-writer` skill
- Export as PDF, host on Google Drive or Notion

## Hosting Options (Free)
| Platform | Best For | Setup Time |
|----------|----------|------------|
| Carrd.co | Single-page portfolio | 30 min |
| Notion public page | Portfolio + case studies | 20 min |
| Google Sites | Quick, professional | 15 min |
| GitHub Pages | Developer credibility | 45 min |

## Portfolio Page Structure
1. **Hero**: Your name + "I help [audience] get [result] using AI"
2. **Services**: 3 cards (Landing Pages, Copywriting, Pitch Decks)
3. **Samples**: 3 portfolio pieces with descriptions
4. **Process**: "How it works" in 3 steps
5. **CTA**: "Get a free sample" button → email/DM link

## Rules
1. Portfolio must be live before any outreach begins.
2. Label spec work honestly — "concept redesign" not "client work."
3. Quality > quantity. 3 great pieces beats 10 mediocre ones.
4. Include turnaround time and pricing on the page.
5. Update portfolio with real client work as it comes in.
