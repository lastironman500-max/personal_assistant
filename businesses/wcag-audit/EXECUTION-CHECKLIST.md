# 5-Day Execution Checklist

**Goal:** $1,000+ revenue in 5 days on a $100 budget
**Start Date:** March 15, 2026

---

## Day 1: Build MVP + Scan 50 Prospects

**Budget Spend:** $12 (domain registration)

### Morning Block (8:00 AM - 12:00 PM): Build Infrastructure

**8:00 - 8:45 AM — Set Up Scanning Workflow**
- [ ] Install axe DevTools Chrome extension
- [ ] Install WAVE Chrome extension
- [ ] Confirm Lighthouse accessible in Chrome DevTools
- [ ] Install Pa11y CLI: `npm install -g pa11y`
- [ ] Test scan on your own website or a known-bad site
- [ ] Bookmark: wave.webaim.org for quick URL scans

**Tools needed:** Chrome browser, Node.js, terminal

**8:45 - 9:45 AM — Build Report Template**
- [ ] Create report template in Google Docs or Notion (use templates/report-template.md)
- [ ] Test filling in template with sample scan data
- [ ] Set up screenshot annotation workflow in Canva (free)
- [ ] Practice: scan a site, screenshot WAVE overlay, annotate in Canva with red boxes

**9:45 - 10:30 AM — Build Cold Email Template**
- [ ] Write 3 cold email variants (use templates/outreach-cold-email.md)
- [ ] Create email signature with name, title, calendar link
- [ ] Set up Calendly (free tier) for 15-minute discovery calls
- [ ] Write mini-audit summary format for embedding in emails

**10:30 - 11:15 AM — Set Up Outreach Tracking**
- [ ] Create Google Sheet: Company | URL | Industry | City | Email | LinkedIn | Issues Found | Outreach Date | Response | Status
- [ ] Install Streak CRM in Gmail (free — tracks email opens)
- [ ] Create Gmail label structure: Prospects / Replied / Interested / Paid / Delivered

**11:15 AM - 12:00 PM — Payment + Marketplace Setup**
- [ ] Confirm Stripe account ready (or PayPal)
- [ ] Create Stripe payment links: $197, $299, $497
- [ ] Register domain ($12) — set up simple landing page (Carrd free or GitHub Pages)
- [ ] Draft Fiverr gig listing (use templates/outreach-fiverr-gig.md)
- [ ] Draft Upwork profile/service listing

### Afternoon Block (12:00 - 6:00 PM): Scan 50 Websites

**12:00 - 1:00 PM — Build Prospect List (Batch 1: 25 sites)**
- [ ] Google search: fashion/clothing stores in NY, FL, CA, IL
- [ ] Google search: restaurants with online ordering in target cities
- [ ] Google search: dentists/chiropractors with "book online" in target cities
- [ ] Record 25 businesses in tracking sheet: domain, name, industry, city
- [ ] Quick filter: sites with modern design (2019+) on Shopify/WooCommerce = high probability targets

**Search strings:**
```
"shopify" clothing store new york
"order online" restaurant miami
dentist "book appointment" chicago
chiropractor "schedule online" los angeles
```

**1:00 - 2:00 PM — Scan First 25 Sites**
- [ ] Open each site, run WAVE extension, screenshot error overlay
- [ ] Record error count and top issue type in tracking sheet
- [ ] Skip sites with 0 errors (rare — move on)
- [ ] Flag sites with 10+ errors as high priority

**2:00 - 3:00 PM — Build Prospect List + Scan (Batch 2: 25 sites)**
- [ ] Repeat for 25 more businesses (different industry or city)
- [ ] Record in tracking sheet

**3:00 - 4:30 PM — Find Email Addresses (Top 30 Prospects)**
- [ ] Hunter.io domain search for each prospect (25 free searches/month)
- [ ] Apollo.io for company + decision-maker lookup (10 free exports/month)
- [ ] LinkedIn: find owner/founder/marketing director for each prospect
- [ ] Check website Contact page, Facebook, Google Business profile as fallbacks
- [ ] WHOIS lookup as last resort

**4:30 - 6:00 PM — Prepare Personalized Outreach**
- [ ] For top 20 email prospects: annotate screenshots in Canva
- [ ] Write personalized first 2 lines for each email referencing specific issues
- [ ] For top 15 LinkedIn prospects: draft DM messages with screenshot
- [ ] Sort by issue severity (highest error count = send first)
- [ ] Queue emails in Gmail drafts

### Evening Block (6:00 - 8:00 PM): Quality Check
- [ ] Verify all 50 sites have issue counts recorded
- [ ] Confirm 30+ email addresses verified
- [ ] Proofread all outreach messages
- [ ] Publish Fiverr gig
- [ ] Test payment links (send yourself a test)

**Day 1 Expected Outcome:** 50 sites scanned, 30 emails found, 20 LinkedIn profiles identified, all outreach drafted
**Contingency if behind:** Cut to 30 sites — prioritize email finding over scan quantity

## Day 2: Outreach Blitz

**Budget Spend:** $25 (email outreach tool — Instantly or Lemlist starter)

### Morning Block (8:00 AM - 12:00 PM): Send First Wave

**8:00 - 9:30 AM — Send 50 Cold Emails**
- [ ] Send in batches of 10 from Gmail (NOT bulk tools on Day 1 — spam risk)
- [ ] Each email includes: their specific issues, annotated screenshot, lawsuit risk data
- [ ] Track sends in Google Sheet
- [ ] Log in Streak CRM for open tracking

**9:30 - 10:30 AM — Send 30 LinkedIn DMs**
- [ ] Send connection requests with personalized notes (15-20 max)
- [ ] For existing connections: send DM with scan screenshot
- [ ] Use templates from outreach-linkedin.md

**10:30 - 11:00 AM — Post Fiverr + Upwork**
- [ ] Publish Fiverr gig (if not done Day 1 evening)
- [ ] Create Upwork service listing
- [ ] Search Upwork for "accessibility audit" jobs posted in last 7 days — submit proposals

**11:00 AM - 12:00 PM — Post on Reddit**
- [ ] r/webdev — educational post about ADA lawsuit trends
- [ ] r/smallbusiness — "Is your website ADA compliant?" value post
- [ ] r/ecommerce — Shopify/WooCommerce compliance thread
- [ ] r/restaurantowners — restaurant-specific exposure post
- [ ] Rule: lead with value/education, mention service only in comments when asked

### Afternoon Block (12:00 - 6:00 PM): Second Wave + Monitor

**12:00 - 2:00 PM — Scan 25 More Websites (New Batch)**
- [ ] Different industry or city than Day 1
- [ ] Record in tracking sheet, find emails

**2:00 - 4:00 PM — Send Second Batch (25 Emails)**
- [ ] Personalize with specific findings
- [ ] Attach annotated screenshots

**4:00 - 5:00 PM — Monitor and Reply**
- [ ] Check email every 30 minutes — respond to replies within 30 minutes
- [ ] For "interested" replies: book Calendly call immediately
- [ ] Do NOT negotiate price over email — get them on a call

**5:00 - 6:00 PM — Check All Channels**
- [ ] Fiverr messages
- [ ] Upwork messages
- [ ] Reddit responses
- [ ] LinkedIn connection accepts + DM replies

**Day 2 Expected Outcome:** 75 emails sent, 30 LinkedIn DMs sent, marketplaces live, Reddit posts up. Target: 3-5 interested replies, 1-2 booked calls.
**Contingency if low opens (<30%):** Subject line problem — switch to alternatives listed in outreach templates

## Day 3: Follow Up + First Deliveries

**Budget Spend:** $0

### Morning Block (8:00 AM - 12:00 PM): Follow-Up Wave

**8:00 - 9:00 AM — Follow Up Day 2 Non-Responders (Email)**
- [ ] Send Day 3 follow-up email to all non-responders (use template in outreach-cold-email.md)
- [ ] Track in Google Sheet

**9:00 - 9:30 AM — Follow Up LinkedIn DMs**
- [ ] For non-responding connections: send screenshot of their issues
- [ ] For new connection accepts: send first DM with scan evidence

**9:30 AM - 12:00 PM — New Outreach Batch (50 New Prospects)**
- [ ] New industry or city (if Day 2 was restaurants in NY, try healthcare in FL)
- [ ] Scan 25 sites, find emails, send personalized emails
- [ ] Send 15 LinkedIn connection requests

### Afternoon Block (12:00 - 6:00 PM): Deliver + Close

**12:00 - 3:00 PM — Deliver Paid Audits**
- [ ] For each paying client: run full scan (all pages, not just homepage)
- [ ] Manual review: homepage, key product/service page, contact form, checkout
- [ ] Fill in report template with findings
- [ ] Export as PDF
- [ ] Email to client via Resend or Gmail

**3:00 - 4:30 PM — Conduct Discovery Calls**
- [ ] For each booked call (15 minutes):
  - [ ] (3 min) Ask about their business
  - [ ] (5 min) Screen share WAVE scan live — show errors on their site
  - [ ] (3 min) Connect issues to lawsuit risk in their industry
  - [ ] (2 min) Close: "Full report is $297, delivered in 48 hours"
  - [ ] (2 min) Handle objection or send payment link

**4:30 - 5:00 PM — Ask First Clients for Referrals**
- [ ] Script: "Do you know other business owners who might have the same concern? 20% referral fee for any client you introduce."

**5:00 - 6:00 PM — Update Everything**
- [ ] Update tracking sheet with all responses
- [ ] Respond to all pending messages
- [ ] Queue tomorrow's outreach

**Day 3 Expected Outcome:** 125 total prospects contacted. First revenue collected. 1-2 deals closed.
**Revenue Target:** $200-$600
**Contingency if $0 revenue:** Switch industries, try $97 "quick scan" offer, post on local Facebook business groups

## Day 4: Scale Outreach + Deliver

**Budget Spend:** $0

### Morning Block (8:00 AM - 12:00 PM): Third Wave

**8:00 - 10:00 AM — New Outreach Batch**
- [ ] 25 new prospects (different industry/city combination)
- [ ] Scan, find emails, send personalized messages
- [ ] LinkedIn: 20 new connection requests + DMs

**10:00 AM - 12:00 PM — Second Follow-Up (Final Touch)**
- [ ] Day 2-3 non-responders: send final follow-up email
- [ ] Subject: "Last thing — {{CompanyName}}.com accessibility check"
- [ ] Create urgency: "I'm fully booked after this week"

### Afternoon Block (12:00 - 6:00 PM): Deliver + Upsell

**12:00 - 3:00 PM — Deliver Outstanding Audits**
- [ ] Complete and deliver all paid reports
- [ ] Quality check: every report has 15+ documented issues with fix recommendations

**3:00 - 5:00 PM — Conduct Calls + Close**
- [ ] Discovery calls with new interested prospects
- [ ] Follow up with anyone who showed interest but hasn't paid

**5:00 - 6:00 PM — Upsell Existing Clients**
- [ ] Offer quarterly re-audit retainer ($97/quarter)
- [ ] Offer remediation implementation (if capable): $297-$497
- [ ] Offer compliance certificate for their website

**Day 4 Expected Outcome:** 150 total prospects contacted. Reports delivered, payments collected.
**Revenue Target:** $600-$900 cumulative
**Contingency if behind:** Call (don't email) all warm leads

## Day 5: Close + Collect + Document

**Budget Spend:** $0

### Morning Block (8:00 AM - 12:00 PM): Close Everything

**8:00 - 9:00 AM — Phone Call Blitz**
- [ ] Call every prospect who showed interest but hasn't paid
- [ ] Script: "We spoke earlier this week about your website's accessibility issues. I wanted to check in before I close out my schedule. Do you have 5 minutes?"

**9:00 - 10:30 AM — Handle Objections**
- [ ] "No budget" → "It's $297. Average ADA settlement is $45,000. Payment plan: $150 today, $147 in 30 days."
- [ ] "Our developer handles this" → "This report is designed to hand off to a developer. Tells them exactly what to fix."
- [ ] "We're too small to be sued" → "85% of ADA web lawsuits target small businesses. Plaintiff attorneys specifically target SMBs."
- [ ] "We'll look into it later" → "I can hold $197 pricing for 48 more hours. Next opening is 2 weeks out."

**10:30 AM - 12:00 PM — Send Invoices + Follow Up**
- [ ] Send Stripe payment links to all committed clients
- [ ] Follow up 1 hour after sending if not paid
- [ ] "The report is complete — wanted to make sure you got the payment link before I send it over"

### Afternoon Block (12:00 - 5:00 PM): Deliver + Document

**12:00 - 3:00 PM — Complete Remaining Audits**
- [ ] Deliver all outstanding reports

**3:00 - 4:00 PM — Confirm Payments**
- [ ] Chase unpaid invoices by phone
- [ ] Verify all received payments in Stripe/PayPal

**4:00 - 5:00 PM — Document Learnings**
- [ ] Which industry had highest response rate?
- [ ] Which subject line got most opens?
- [ ] Which objection came up most often?
- [ ] Which city had most interested prospects?
- [ ] What issue type created most urgency in calls?
- [ ] What did each closed client say that tipped them toward buying?
- [ ] What would you do differently in Week 2?

**Day 5 Expected Outcome:** All deals closed, all reports delivered, all payments collected.
**Revenue Target:** $1,000+ cumulative
**Contingency:** "Closing my books today" urgency messaging to all warm leads
