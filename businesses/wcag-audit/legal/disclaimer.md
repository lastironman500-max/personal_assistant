## 1. Report Disclaimer Text

> This disclaimer appears on the cover page and executive summary of every report.

### Cover Page Disclaimer

**IMPORTANT NOTICE**

This document is an **Automated Accessibility Scan Report** generated using axe-core (an industry-standard automated testing engine) and AI-powered analysis (Claude by Anthropic). It is NOT a full manual accessibility audit, legal compliance certification, or guarantee of conformance with any law or regulation.

**What this report is:**
- A technical scan identifying programmatically-detectable accessibility barriers
- An AI-generated interpretation of findings with remediation guidance
- A prioritized list of issues found by automated tools

**What this report is NOT:**
- A legal opinion or legal advice
- A guarantee of ADA, Section 508, EAA, or any other regulatory compliance
- A substitute for manual testing with assistive technologies
- A certification of WCAG conformance
- A guarantee against legal action, demand letters, or lawsuits

**Scope limitation:** Automated accessibility testing tools detect approximately 57% of potential WCAG 2.1 issues by volume (source: Deque Systems published research). The remaining 43% of potential issues — including screen reader reading order, cognitive accessibility, focus trap behavior, meaningful alt text quality, error recovery flows, timing/animation issues, and custom widget behavior — require manual testing by trained accessibility specialists using assistive technologies.

**AI disclosure:** This report was generated with the assistance of artificial intelligence (Claude by Anthropic). AI was used to deduplicate findings, write plain-English descriptions, generate remediation code examples, and prioritize issues. All underlying scan data was produced by axe-core, an open-source automated testing engine.

**Recommendation:** This report should be used as a starting point for accessibility remediation. For complete WCAG conformance assessment, we recommend engaging a certified accessibility specialist to conduct manual testing with screen readers (NVDA, JAWS, VoiceOver), keyboard navigation testing, and user testing with people with disabilities.

### In-Report Footer Disclaimer (Every Page)

*This automated scan report is for informational purposes only. It does not constitute legal advice or a guarantee of regulatory compliance. Automated tools detect ~57% of WCAG issues. Manual testing is recommended for complete conformance assessment.*
