---

## Service Description

We sell **Automated WCAG Accessibility Scan Reports**. A business owner provides their website URL. Within minutes, they receive a branded PDF report documenting every programmatically-detectable accessibility violation, its legal risk, and exactly how their developer should fix it — prioritized by severity.

**What the pipeline does (zero human involvement):**

1. **Crawl** — Playwright discovers all pages via sitemap + link-following
2. **Scan** — axe-core tests every page against WCAG 2.2 AA criteria
3. **Analyze** — Claude API deduplicates, writes plain-English explanations, generates code-level fixes, produces a prioritized remediation roadmap
4. **Generate** — Puppeteer renders a professional PDF report
5. **Deliver** — Resend emails the report to the client

**Critical positioning:** This is an "Automated Accessibility Scan Report" — never a "full audit," "compliance certification," or "guaranteed ADA compliance." Every report discloses the 57% automated detection ceiling and recommends manual expert review for complete conformance.
