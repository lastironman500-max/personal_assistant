## FAQ

**Q: Is this a full manual accessibility audit?**
A: This is an automated scan using industry-standard tools (axe-core), enhanced with AI analysis. It detects approximately 57% of WCAG issues — the programmatically-detectable ones. The report clearly identifies what requires additional manual testing. For most businesses that have never done any accessibility testing, this is the critical first step.

**Q: Will this guarantee I won't get sued?**
A: No accessibility scan or audit — automated or manual — can guarantee you won't receive a demand letter or lawsuit. What this report does is identify your highest-risk issues so you can fix them proactively, significantly reducing your exposure.

**Q: What tools do you use?**
A: axe-core (the industry-standard accessibility engine maintained by Deque, used by Google, Microsoft, and the BBC), Playwright for page rendering, and Claude AI for analysis and remediation guidance.

**Q: How is this different from a free tool like WAVE or Lighthouse?**
A: Free tools produce raw technical data. This report translates findings into plain-English explanations a business owner can understand, prioritizes issues by legal risk, provides developer-ready fix instructions with code examples, and delivers a professional PDF you can hand to your lawyer, board, or web developer.

**Q: Can you fix the issues you find?**
A: This service focuses on identification and remediation guidance. I can recommend trusted developers for implementation, or message me about a custom remediation package.

**Q: Do you offer ongoing monitoring?**
A: Yes — message me about monthly re-scan packages ($497/month) that track your compliance progress over time and catch regressions introduced by code changes.

**Q: What if my site is built with React/Angular/Vue (SPA)?**
A: The scanning tools use a full Chromium browser that renders JavaScript, so Single Page Applications are fully supported.
