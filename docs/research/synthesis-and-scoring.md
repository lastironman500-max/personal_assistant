The winning formula: **Automated Tool (data) + LLM (narrative) = Sellable Report**

## Deep Dive: Top 3

### #1: WCAG Accessibility Audit Reports (Score: 56)

**What:** AI scans a website for WCAG 2.1/2.2 compliance issues using axe-core, generates a professional audit report with findings, severity, and remediation steps.

**Why it wins:**
- **Legal mandate** — EU Accessibility Act (EAA) took effect June 2025, ADA lawsuits hitting 4,000+/year in US. Non-discretionary spend.
- **Existing price**: Manual audits cost $1,250-$15,000. Automated tools (axe-core) are free but produce raw JSON.
- **Our price**: $200-500 per report. 80-95% cheaper than agencies.
- **AI does 100%**: axe-core scans → Claude writes narrative report with plain-English fixes → PDF delivered
- **Every business with a website is a prospect** — millions of SMBs need this
- **Zero competition at this price point** — agencies charge $1,250+, tools give raw data

**Revenue path to $1,000:**
- 2-5 reports at $200-500 each
- Cold outreach to businesses with visible accessibility issues (detectable by scanning)

**Tools needed:** axe-core (free), Playwright (free), Claude API (~$5), PDF generator (free)

---

### #2: Full Website Audit Bundle (Score: 53)

**What:** Combine SEO + Performance + Accessibility + Security into one comprehensive report. Charge $300-800.

**Why it's strong:**
- Higher ticket than single-category audits
- Same tools, more value, more defensible
- Businesses love "one-stop" solutions

**Risk:** More complex to build, slower to first sale

---

### #3: SEO Audit Reports (Score: 50)

**What:** Crawl website, analyze SEO issues, generate client-ready report with prioritized fixes.

**Why it's strong:**
- Massive existing market (Ahrefs $100M+ ARR proves demand)
- Screaming Frog + Claude = complete report in minutes
- Every agency and SMB buys SEO audits

**Risk:** More competitive than accessibility niche

## Recommendation

**Start with #1: WCAG Accessibility Audit Reports**

Reasons:
1. Legal mandate = non-discretionary spend (strongest demand signal)
2. Widest pricing gap ($200 vs $1,250+ for same deliverable)
3. 100% AI-automatable (axe-core + LLM narration)
4. Easy to find prospects (scan websites, find issues, reach out with proof)
5. Expandable to full website audit bundle (#2) after validation
